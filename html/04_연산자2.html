    - 좌변, 우변을 같다, 같지 않다, 초과, 미만, 이상, 이하를 비교해서
      맞을 경우 true, 아닐 경우 false를 반환한다!!!
      (boolean == 논리형 == true / false)

%(나머지), 비교 연산자를 이용해서 짝수 / 홀수, 배수 / 약수 확인

복합 대입 연산자(+=, -=, *=, /=, %= )

    - 기존 변수에 저장되어 있는 값에 
      새롭게 대입되는 값의 산술 연산결과를 대입하는 연산자

    - 코드가 짧고, 속도 우위를 가짐

    ex) let temp = 10;
        temp += 5;    // temp = temp + 5;

논리 연산자(&&, ||, !)

    1. && : (AND) 연산자 -> A && B
      - A 와 B 모두 true 인 경우에만 
        A && B 결과가 true 인 연산자 (나머진 false)
      
      - 글 / 말 / 문장에서 
        ~와, 그리고(이고), 이면서(면서), ~까지, ~부터, ~사이 
        라는 글귀가 있으면 AND 연산을 작성

      ex) 104 는 100 이상의 숫자이면서 짝수인가?
          -> (104 >= 100) && (104 % 2 === 0)  -> true!!   
  
    2. || : (OR) 연산자 -> A || B  
      - A 와 B 모두 false 인 경우에만 
        A && B 결과가 false 인 연산자 (나머진 false)
  
      - 글 / 말 / 문장에서 
        또는, ~이거나(거나) 글귀가 있으면 OR 연산을 수행
    
      ex) 4는 10을 초과하거나 짝수인가?
          -> (4 > 10) || (4 % 2 === 0) -> true 
  
    3. !(NOT) 연산자
       - 논리값을 반대로 바꾸는 연산자

         !true  -> false
         !false -> true

       ex) !(true && false) -> true
           (10 > 5) || !(10 % 2 === 0) -> true 
           // 앞부분이 true 이기 때문에 후자를 계산하지 않아도 알 수 있음
           // 실제 컴퓨터는 앞부분만 계산 후 종료
